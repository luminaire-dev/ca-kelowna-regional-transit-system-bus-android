Full description:
This app adds Kelowna Regional Transit System (BC Transit) buses information to MonTransit.

The app contains the planned schedule as well as real time predictions from NextRide and news from YouTube and @BCTransit on Twitter.

Kelowna Regional Transit System buses serve Kelowna, Lake Country, Peachland in British Columbia, Canada.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by BC Transit.
http://bctransit.com/*/footer/open-data

This application is free and open-source:
https://github.com/mtransitapps/ca-kelowna-regional-transit-system-bus-android

This app is not related with BC Transit and Kelowna Regional Transit System.
